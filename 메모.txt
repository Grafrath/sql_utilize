✅ Step 1: 데이터베이스 연결 생성

1. 상단 메뉴 → **Database → New Database Connection**
2. 검색창에 `mysql` 입력
3. "MySQL" 선택 → **Next*

✅ Step 2: 연결 정보 입력
Host	localhost
Port	3306 (MySQL 기본 포트)
Username	root
Password	(MySQL 설치할 때 만든 비밀번호)

sql문장 사용한다

스크립트에 CREATE DATABASE 명령어 입력 ex)CREATE DATABASE testdb

CREATE USER 명령으로 유저 생성 ex)CREATE USER 'user'@'%' IDENTIFIED BY 'password';

GRANT 명령으로 권한 부여 ex)GRANT 권한 ON testdb.* TO 'user'@'%';

SHOW 명령으로 권한 확인 ex) SHOW GRANTS(현재 로그인한 사용자 권한 보기) SHOW GRANTS FOR 'testuser'@'%'(특정 사용자 권한 보기)


테이블의 생성 CREATE
테이블의 삭제 DROP
테이블 변경 ALTER
